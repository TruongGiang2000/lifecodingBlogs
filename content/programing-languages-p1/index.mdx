## JavaScript.

<div style={{lineHeight:2}}>
- Có thể nói JavaScript đang là ngôn ngữ phổ biến nhất hiện nay theo bình chọn của Stack Overflow. "Javascript là một ngôn ngữ lập trình dựa trên nguyên mẫu với cú pháp phát triển từ C. Giống như C , javascript có khái niệm từ khoá, do đó Javascript gần như không thể mở rộng" (theo wikipedia). Tại sao Javascript lại phổ biến đến vậy, không nói ngoa chứ javascript có thể làm ra hầu hết mọi thứ. Bắt đầu từ ứng dụng trên nền tảng mobile thì React Native là một framework khá phổ biến dùng để lập trình ứng dụng đa nền tảng đang sử dụng javascript, nó có thể tạo ra được Android app, IOS app, và cả UWP (Window app). Ngoài ra javascript còn có thể dùng để tạo ra một trang web chuyên nghiệp. Cuối cùng javascript được dùng để viết server (backend) thông qua Nodejs. Những bạn nào muốn trở thành một fullStack thì hãy bắt đầu với javascipt nào ! Còn những bạn muốn đi chuyên một nền tảng nào đó thì cũng có thể bắt đầu với javascript nhé.
</div>

## Ngôn ngữ C.
<div style={{lineHeight:2}}>
- Ngôn ngữ lập trình C là một ngôn ngữ mệnh lệnh được phát triển từ đầu thập niên 1970 bởi Dennis Ritchie để dùng trong hệ điều hành UNIX. Từ đó, ngôn ngữ này đã lan rộng ra nhiều hệ điều hành khác và trở thành một trong những ngôn ngữ phổ dụng nhất. Đa phần các ngôn ngữ lập trình hiện tại đều dựa trên C để viết ra nên C cũng là một khởi đầu tốt để các bạn có thể học thêm nhiều ngôn ngữ khác và chọn ra bến đỗ của đời mình. Vậy học C làm được gì ? Ngôn ngữ lập trình C dùng để phát triển các hệ thống kết nối với phần cứng ngoài ra thì C còn đóng vai trò quan trọng trong việc phát triển các loại ngôn ngữ mới. C còn có 2 người anh đó là C # và C plus plus được dùng phổ biến trong việc viết server và tạo ra những con game cuốn hút. Các hệ điều hành phổ biến hiện nay như Microsoft, Linux,... đa phần đều dùng C để viết.
</div>

## Java.
<div style={{lineHeight:2}}>
- Ngôn ngữ lập trình đầu tiên mình được học một cách chuyên sâu đó chính là Java. Đây là một ngôn ngữ lập trình hướng đối tượng, dựa trên lớp được thiết kế để có càng ít phụ thuộc thực thi càng tốt. Nó là một ngôn ngữ lập trình có mục đích chung cho phép các nhà phát triển ứng dụng viết một lần, chạy ở mọi nơi, nghĩa là mã Java đã biên dịch có thể chạy trên tất cả các nền tảng hỗ trợ Java mà không cần biên dịch lại. Java là ngôn ngữ mà bạn khó có thể bỏ qua nếu bạn là một lập trình viên Android Mobile, mặc dù vào năm 2017 Kotlin đã được Google hỗ trợ đầy đủ để lập trình trên hệ điều hành Android nhưng đa phần các công công ty công nghệ tại Việt Nam hiện nay vẫn còn sử dụng Java để xây dựng ứng dụng Android nên các bạn không phải lo Java lỗi thời. Bên cạnh đó Java còn có thể viết được cả server và cũng khá phổ biến trong giới lập trình. Nhìn chung đây là một ngôn ngữ đáng để quan tâm nếu bạn muốn trở thành một Android Delevoper hay một Backend thu hút.
</div>

## Python.
<div style={{lineHeight:2}}>
- Thật trùng hợp nhưng Python cũng là tên của con rắn khổng lồ trong thần thoại hy lạp, nhưng theo mình biết thì cái tên Python được tác giả lấy từ tên trong một chương trình hài "Monty Python's". Python là một ngôn ngữ lập trình bậc cao cho các mục đích lập trình đa năng, do Guido van Rossum tạo ra và lần đầu ra mặt vào năm 1991. Đây là ngôn ngữ dành cho những bạn nào đi theo trí tệ nhân tạo hay còn gọi là __AI__. Một ngôn ngữ khá phổ biến trong giới Trí Tuệ Nhân Tạo. Nếu như các bạn tìm hiểu thì cũng biết rằng AI cũng đang rằng phát triển mạnh theo từng năm và dự đoán sẽ là xu thế trong một vài năm tới. Vậy tại sao không bắt đầu với Python nào.
